# موقع الدعم للقضية الفلسطينية

موقع مخصص لدعم القضية الفلسطينية من خلال عدة قنوات:
- الدعاء
- المقاطعة
- التبرع
- المنتجات البديلة

## المميزات
- تصميم عصري وجذاب
- واجهة مستخدم سهلة الاستخدام
- متوافق مع جميع الأجهزة
- دعم كامل للغة العربية

## التقنيات المستخدمة
- HTML5
- CSS3
- JavaScript
